# Milestone 0 – Cross‑Cultural Collaboration

## Summary

This milestone established communication norms, expectations, constraints, and learning goals for the ELO2 Gold Pathfinder ML Project.

## Group Norms

- Respectful, inclusive communication  
- Weekly meeting cadence  
- Consensus-based decision making  
- Clear accountability and shared responsibility  

## Communication Plan

Primary tools: GitHub, WhatsApp/Slack, Zoom, Google Docs.  
Weekly 90‑min sync + optional mid‑week check‑in.

## Constraints

- Time: Sept 22–30  
- Technical: open‑source tools only  
- Skill: mixed geoscience/data backgrounds  
- Mitigation: pair-learning, reproducible workflows  

## Learning Goals

- Strengthen Python + data visualization  
- Apply geochemical reasoning to data science  
- Improve communication + project documentation  

## Reflection

This milestone improved structure, alignment, and readiness for collaborative work.
